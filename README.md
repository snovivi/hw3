# Опрос "Heritage Tourism"
Данный опрос проводился с целью выявление заинтересованности современной молодежи в поесщении культурных мест.Но помимо этого, мне удалось выйяснить их основные цели путшествий, предпочтения в путшествии (транспорт, люди с которыми они отправились бы в путшествие, страна). В опросе приняли участие **34** человека. 
Данный вид опроса полезен туристическим комапниям, ведь они могли бы контролировать спрос на то или иное путешествие, грамотно формировать турстические пакеты и варьировать цены на туристические поездки.
